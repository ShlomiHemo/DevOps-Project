# DevOps Project

A complete **DevOps pipeline** for a Java JSP web app, including automated deployment, CI/CD, performance testing, and monitoring.

- **CI/CD & Deployment:** Jenkins pipelines automating build, test, and deployment on Apache Tomcat; public exposure via Zrok.

- **Performance & Load:** Gatling tests up to 120 concurrent users; system stability and stress-tested behavior analyzed.
