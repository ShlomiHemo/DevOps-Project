# DevOps Project

A complete **DevOps pipeline** for a Java JSP web app, including automated deployment, CI/CD, performance testing, and monitoring.

---

## ðŸ”¹ Key Highlights

- **CI/CD & Deployment:** Jenkins pipelines automating build, test, and deployment on Apache Tomcat; public exposure via Zrok.  
- **Automated Testing:** End-to-end UI tests with Selenium IDE, including multi-page navigation and form submission.  
- **Performance & Load:** Gatling tests up to 120 concurrent users; system stability and stress-tested behavior analyzed.  
- **Monitoring & Availability:** Custom scripts track uptime, HTTP status, and integration with CI/CD pipelines.  
- **Skills Demonstrated:** CI/CD, automated testing, monitoring, performance benchmarking, deployment automation, DevOps problem-solving.
